# Painel de Clima em Tempo Real

Aplicação web que consulta a API da OpenWeather para mostrar informações de clima de uma cidade.

Usei as tecnologias: HTML5, CSS3, JavaScript, API OpenWeather.

Esse painel interativo permite que o usuário busque a cidade desejada e veja o clima em tempo real e as condições meteorológicas.

O objetivo do projeto é praticar o consumo de API, manipular o DOM, e usar lógica de programação front-end.

Como rodar o projeto: Clone o repositório, crie um arquivo `config.js`, e adicione sua chave da API da OpenWeather (https://openweathermap.org/api).
